# Ansible Role: marvel-nccr.yambo

An ansible role that installs [yambo](http://www.yambo-code.org/) on Ubuntu.

## Installation

`ansible-galaxy install marvel-nccr.yambo`

## Role Variables

See `defaults/main.yml`

## Example Playbook

```
  - hosts: servers
    roles:
    - role: marvel-nccr.yambo
```
